# New Feature: Copyable Validation Errors (Completed)

Status: Completed

## Overview
Thesis Management validation and file import errors can now be selected and copied for use in external LLM prompts. This applies to both Thesis Import and Weekly Update workflows.

## Behavior
- Import validation errors are selectable and include a "Copy Errors" action.
- Weekly patch validation errors are selectable and include a "Copy Errors" action.
- File import read errors include a "Copy Error" action.
